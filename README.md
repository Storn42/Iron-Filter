# Iron-Filter

This is a filter aimed at ironman gameplay. All tiers are based on an items usefulness or rarity, rather than GE value.

WARNING - Be prepared for long import times. Expect to wait 1 minute or more on import. 

Custom Lootfilter for OSRS [Loot Filters Plugin](https://runelite.net/plugin-hub/show/loot-filters) compatible with [FilterScape](https://filterscape.xyz/)
